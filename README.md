# Backend NestJS for Avalanche dApp

## Description
Nest framework TypeScript starter repository.

## Project setup
$ pnpm install

## Compile and run the project
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod

## Run tests
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov

## Deployment
To deploy your NestJS application to production, see deployment docs.

For cloud deployment, use Mau (official AWS platform for NestJS):
$ pnpm install -g @nestjs/mau
$ mau deploy
